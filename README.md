# Portfolio de Francisco Rodríguez

Portfolio personal construido como sitio estático para presentar perfil, stack técnico y proyectos.

## Objetivo

Mostrar un perfil junior orientado a backend y desarrollo web con una presentación propia, responsive y mantenible.

## Stack del sitio

- HTML
- CSS
- JavaScript
- Font Awesome para iconografía

## Contenido

- Presentación profesional
- Perfil y formación
- Stack técnico
- Proyectos destacados
- Datos de contacto
